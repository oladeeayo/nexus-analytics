Process

Up and running in minutes

No data engineering required. Install once, analyze forever.

01

Install the SDK

One script tag or npm install. Works with React, Vue, Angular, vanilla JS — and every mobile platform.

// Install via npm
npm install @amplitude/analytics-browser

// Initialize
amplitude.init('YOUR_API_KEY');
02

Track what matters

Fire events with any properties you care about. Nexus automatically enriches them with device, location, and session data.

amplitude.track('CTA Clicked', {
  button: 'Sign Up',
  section: 'Hero',
  plan: 'Pro'
});
03

Build charts in seconds

Drag-and-drop funnel builder. Cohort creator. Retention heatmaps. No SQL needed — but available if you want it.

04

Ship with confidence

Set up alerts for metric regressions, share dashboards with stakeholders, and make every release data-driven.
